## Purpose
Contain a list of all templates with some extra information

## Content

### How to create a new feature documentation
1. Duplicate the following template(s) to their corresponding `Drafts/` directory.
2. The ADR should have a default status of `Proposed`
3. Fill in the ADR and feature documentation.
4. Once completed, make a PR for the documentation.
5. Implement any feedback
6. Once the PR is approved, move the files out of the `Drafts/` directory
7. Do not forget to change the ADR status to `Accepted`.
